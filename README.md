# FORMAPP Java Web Application

A Java Servlet based web application for handling HTML form data using Apache Tomcat.

This project demonstrates form submission and request handling using Java Servlets.

## Features
- HTML form handling
- Servlet-based request processing
- User input handling
- Dynamic response generation

## Technologies Used
- Java
- Servlet API
- HTML5
- Apache Tomcat

## Project Structure

FORMAPP/
│
├── form.html
│
└── WEB-INF/
     ├── web.xml
     └── classes/
          └── FormServlet.java

## How to Run

1. Install Apache Tomcat
2. Copy project into Tomcat `webapps` folder
3. Start Tomcat server
4. Open application in browser

Example:

http://localhost:8080/FORMAPP

## Learning Outcomes
- Form data processing
- Handling HTTP requests and responses
- Java Servlet lifecycle
- Web application deployment using Tomcat
